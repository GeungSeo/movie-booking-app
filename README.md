# Movie Booking App

Next.js 기반의 영화 예매 시스템입니다.  
Vercel에 배포되어 있으며, TMDB API를 연동해 인기 영화 정보를 제공합니다.

## 배포 링크
https://movie-booking-gobigm3lz-eunseokas-projects.vercel.app/

## 주요 기능
- 인기 영화 목록 표시 (TMDB API 연동)
- 영화/극장/상영 정보 등록 (관리자)
- 예매 및 예매 내역 확인

## 사용 기술
- Next.js 14 (App Router)
- TMDB API
- Vercel (배포)
- Docker & Nginx 

## 실행 방법
```bash
npm install
npm run dev
